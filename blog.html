Difference between local storage and session storage.

Local storage and session storage both store a user's data. But the main difference is that if any data is stored in session storage, that data is no longer available after closing that tab. And if any information is saved in local storage, even if the data is closed, even if the browser is closed, the user's data is saved in local storage.
